Prueba .NET

Realizar una api que tenga como base de oracle o SQL de preferencia, o la Base que
prefiera y tenga y sea de tipo relacional, que tenga tablas de profesores, estudiantes,
materia y notas en el cual se pueda hacer crud completo en todas las tablas menos la de
notas que solo puede hacer post, put y get, debe retornar el endpoint objetos en json y que
lo carguen en un link público en github

para tener en cuenta:
1. La base de datos de preferencia oracle, no puede ser NoSQL
2. Mínimo se debe tener el crud completo de estudiantes, profesores y materias
3. Debe entregar un bosquejo de como hizo el diagrama ER para poder hacer la evaluación
de la funcionalidad
4. La evaluación será 30% profesores, 30% estudiantes, 10% materias, 10% notas, 10%
diagrama ER y 10% elementos adicionales
5. Profesores y estudiantes mínimo con 6 campos cada tabla
6. No se puede mostrar ningún id de la BD solo datos proporcionados por el usuario en los
objetos json
7. Para el CRUD debe tener getAll(), getById(), save(), update(), delete()
8. De preferencia dejar comentarios que hagan saber la funcionalidad del codigo que estara
dentro del 10% de elementos adicionales
